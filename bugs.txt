# Jarvis Bug Tracker

## 1. Sensor/Hardware Bugs
- [ ] **Temperature Check**: Jarvis reports "Mac running at 0.0°C" even during heavy load. 
  - *Cause*: `osx-cpu-temp` missing and `powermetrics` needs sudo.
- [ ] **System Diagnostics**: Some system stats are not reporting correctly on Mac Silicon.

## 2. UI & Experience Bugs
- [ ] **Thinking UI "New Screen"**: The Tkinter popup steals focus or appears as a new window/screen on macOS instead of a silent overlay.
- [ ] **News Briefing Sync**: Main listener starts *while* Jarvis is still reading the news, leading to "No speech detected" loops.

## 3. Intelligence & Routing Bugs
- [ ] **Visual vs Brightness**: "Tell me what's on my screen" matches to `brightness_up` instead of `take_screenshot`. (Matched `screen brighter` with 0.48 conf).
- [ ] **Research vs OpenApp**: "Tell me all the details about X" matches to `open_app` (trying to open "X") instead of using the Research agent/ManagerBrain.
- [ ] **Greeting Timeout**: Silence threshold is too sensitive during low-volume greetings. ("No speech detected" loop).
- [ ] **Intent Confidence Floor**: Some conversational queries are hitting the "Safety: gemini — Below minimum confidence floor" trigger too easily.
